| `Code`                                       | **string*                                    | :heavy_minus_sign:                           | The code identifier for the Technology Area. |
| `ID`                                         | **int64*                                     | :heavy_minus_sign:                           | Unique identifier for the Technology Area.   |
| `Name`                                       | **string*                                    | :heavy_minus_sign:                           | The name of the Technology Area.             |
